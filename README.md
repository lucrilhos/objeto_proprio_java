# Projeto: Modelagem de Objeto em Java - Jaqueta com Zíper

## Descrição

Este projeto foi desenvolvido com o objetivo de praticar os conceitos de **Programação Orientada a Objetos (POO)** em Java, utilizando a IDE Eclipse.

A proposta consiste na modelagem de um objeto do mundo real — uma **Jaqueta com Zíper** — aplicando atributos, métodos e comportamentos dentro de uma estrutura organizada em packages.
(zipado por ser um projeto pequeno e alguns problemas no meu CMD)

###########################

## Conceitos Aplicados

* Classe e Objeto
* Atributos privados invés de publicos (encapsulamento)
* Construtor
* Métodos e comportamento 
* Packages 

###########################

## Estrutura básica do projeto

```
src
 ├── br.com.fiapmodel
 │     └── JaquetaZiper.java
 │
 └── br.com.fiapride.main
       └── SistemaPrincipal.java
```

###########################

## Classe JaquetaZiper

A classe representa uma jaqueta com os seguintes atributos:

* Cor
* Tamanho
* Material
* Estado/situação do zíper
E comportamentos como:
* Abrir zíper
* Fechar zíper
* Exibir detalhes

###########################

# SistemaPrincipal

O que faz:
* Cria instâncias da classe `JaquetaZiper`
* Executa os métodos
* Simula o comportamento do objeto
